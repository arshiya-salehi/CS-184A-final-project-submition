* `DAVIS/` csv datasets split into training, validation, and testing
* `histories/` JSON files containing logs of training and evaluation conducted on a model
* `models/` PT files holding the saved state of the best epoch created during training for baseline and duo models
* `src/conplex-dti/` original src code used by research paper authors
* `src/architectures.py` a collection of model architectures we wrote for this project
* `src/evaluation.ipynb` notebook used to evaluate models on a DAVIS testing dataset
* `src/run_contrastive.sh` example of terminal command used to train models with the in-build CLI
* `project.html` export of project.ipynb
* `project.ipynb` demonstration of the performance of different models investivated through data visuals
* `README.md` "contains a 1 line description of each file in project/"
* `requirements.txt` python dependencies to run project.ipynb
